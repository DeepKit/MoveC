## 📖 测试示例

### 示例 1：数据库持久化测试

```pascal
procedure TestTaskPersistence;
var
  Task: uSyncDatabase.TSyncTask;
  TaskID: Integer;
  Database2: TSyncDatabase;
begin
  // Arrange - 创建任务
  FDatabase.Connect;
  Task.Name := 'Persistent Task';
  Task.SourcePath := 'C:\Source';
  Task.TargetPath := 'D:\Target';
  TaskID := FDatabase.CreateSyncTask(Task);
  
  // 关闭数据库
  FDatabase.Disconnect;
  FreeAndNil(FDatabase);
  
  // Act - 重新打开数据库
  Database2 := TSyncDatabase.Create(FTestDbPath);
  Database2.Connect;
  RetrievedTask := Database2.GetSyncTask(TaskID);
  
  // Assert - 验证
  Assert.AreEqual('Persistent Task', RetrievedTask.Name);
end;
```

**如果持久化失败，测试输出：**
```
[TestTaskPersistence] FAIL (12 ms)
  Expected: 'Persistent Task'
  Actual  : ''
  Message : 重新打开后任务应该仍然存在
```

✅ **立即发现问题！**

## 🎓 如何添加新测试

### 发现新Bug时

**不要手工测试！应该：**

```pascal
[Test]
procedure TestMyBugFix;
begin
  // Arrange - 重现Bug的场景
  // Act - 执行操作
  // Assert - 验证修复后的行为
end;
```

### 示例：测试任务编辑不保存

```pascal
[Test]
procedure TestTaskEditPersistence;
var
  Task: TSyncTask;
  TaskID: Integer;
begin
  // Arrange
  Task := TSyncTask.CreateWithDatabase(FDatabase);
  Task.Name := 'Original';
  Task.Save;
  TaskID := Task.TaskID;
  
  // Act - 编辑
  Task.Name := 'Edited';
  Task.Save;  // 这行如果有Bug就会失败
  
  // Assert - 重新加载验证
  Task.Free;
  Task := TSyncTask.CreateWithDatabase(FDatabase);
  Task.TaskID := TaskID;
  Task.Load;
  
  Assert.AreEqual('Edited', Task.Name);
end;
```
