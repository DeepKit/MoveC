- **主字体**: Microsoft YaHei UI (现代、清晰)

- **大小层次**: 18px(标题) → 10px(正文) → 9px(辅助) → 8px(状态)

### 交互设计
- **主题切换**: 一键切换浅色/深色主题
- **智能颜色**: 状态消息根据主题自动调整颜色
- **平滑过渡**: 所有界面变化都有平滑过渡效果
- **响应式反馈**: 用户操作的即时视觉反馈
