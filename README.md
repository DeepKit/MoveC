# C盘瘦身工具 v2.0 企业级版本 🚀

## 📖 项目简介

C盘瘦身工具是一个功能强大的Windows系统磁盘空间优化工具，集成了智能目录迁移、零决策重复文件清理、企业级配置管理等先进功能。通过AI驱动的智能分析，帮助用户安全高效地释放C盘空间。

## ✨ 核心特性

### 🤖 零决策智能重复文件清理 ⭐ **新功能**
- **智能决策算法**: 多维度评分系统，自动选择最佳保留文件
- **一键式操作**: 扫描→分析→清理，全程自动化
- **三种清理模式**: 保守/标准/激进，满足不同需求
- **安全保障**: 白名单保护、智能回收站、渐进式清理

### 🔍 企业级依赖关系分析 ⭐ **新功能**
- **智能风险评估**: 迁移前自动分析注册表依赖
- **四级风险等级**: 低风险/中等风险/高风险/极高风险
- **详细分析报告**: 依赖项详情、影响评估、安全建议
- **用户友好提示**: 根据风险等级提供相应的操作建议

### ⚙️ 配置管理系统 ⭐ **新功能**
- **图形化配置界面**: 工具菜单→配置管理器
- **多类别设置**: 常规、迁移、清理、高级四大类
- **实时配置应用**: 主题切换、语言设置即时生效
- **配置持久化**: 自动保存用户偏好设置

### 🚀 高性能文件操作引擎 ⭐ **新功能**
- **多线程并行处理**: 可配置线程数量，显著提升复制速度
- **智能缓冲管理**: 可调节缓冲区大小，优化内存使用
- **压缩和加密支持**: 企业级数据安全保障
- **自动回退机制**: 高性能引擎失败时自动使用标准方式

### 📊 智能目录迁移
- **完整迁移流程**: 复制→备份→符号链接→验证→回滚保护
- **符号链接优化**: 优先Junction，失败自动降级Symlink
- **一致性校验**: 文件数量和大小双重验证
- **智能进度显示**: 实时显示当前文件和传输进度

### 🧹 系统清理功能
- **多目标清理**: 回收站、临时文件、Windows更新缓存
- **智能备份清理**: 一键清理历史备份，释放额外空间
- **安全删除**: 默认移动到回收站，支持永久删除选项

### 🎨 现代化用户界面
- **深色/浅色主题**: 支持主题切换，保护视力
- **多语言支持**: 20种语言界面，全球化体验
- **响应式设计**: 适配不同分辨率和DPI设置
- **直观操作**: 双面板设计，拖拽式目录选择

## 🚀 快速开始

### 系统要求
- Windows 7/8/10/11
- 管理员权限（推荐）
- 至少100MB可用磁盘空间

### 安装步骤
1. 下载最新版本的DiskCleanup.exe
2. 以管理员身份运行程序
3. 开始使用各项功能

### 基本使用流程
1. **浏览目录**: 双击左侧目录树选择要分析的目录
2. **计算大小**: 点击"计算目录大小"按钮了解空间占用
3. **分析可行性**: 点击"分析并标记"按钮评估移动风险
4. **执行操作**: 根据分析结果决定是否移动目录

## 📋 功能详解

### 目录浏览
- **双击切换**: 双击目录树快速切换并显示文件
- **路径同步**: 自动更新对应的路径编辑框
- **文件列表**: 实时显示目录中的文件内容

### 大小计算
- **递归计算**: 包含所有子目录的完整大小统计
- **性能优化**: 对根目录等大目录给出性能警告
- **详细显示**: 在标签和状态区域显示格式化结果

### 可行性分析
- **智能判断**: 基于目录性质判断符号链接可行性
- **风险分级**: 
  - 🟢 **可链接**: 安全移动，不影响程序运行
  - 🟡 **有风险**: 可能影响程序，但符号链接通常可解决
  - 🔴 **禁止移**: 严重影响系统，禁止移动
- **详细说明**: 提供具体的判断原因和操作建议

## 🛡️ 安全注意事项

### 使用前须知
1. **备份重要数据**: 在执行任何移动操作前，请备份重要文件
2. **管理员权限**: 建议以管理员身份运行，确保有足够权限
3. **系统目录**: 绝对不要移动Windows系统目录
4. **测试环境**: 重要系统建议先在测试环境验证

### 风险提示
- **符号链接**: 某些老旧程序可能不支持符号链接
- **权限问题**: 移动系统文件可能导致权限问题
- **路径依赖**: 硬编码路径的程序可能受到影响
- **性能影响**: 跨磁盘的符号链接可能影响访问速度

## 🚀 最小可交付版本（MVP）

- 目录迁移（C→D）：复制→备份→原址创建链接（默认目录联接 /J，失败回退符号链接）→成功提醒测试与清理备份→失败回滚
- 清理功能：清空回收站、清理临时文件、清理 Windows 更新缓存
- 分析（同步版）：目录总大小、TopN 大文件、类型聚合（跳过 reparse point）
- 非阻塞启动：FormShow → TTimer 一次性延时初始化；迁移阶段提供进度条与“停止”取消

## ✅ 已采纳的默认行为
- 复制后校验不一致时：提示+可继续（需用户确认）
- 链接创建优先：目录联接（/J），符号链接为回退
- 一键删除最近备份：在“清理”菜单提供入口（执行前有提醒，移动到回收站）
