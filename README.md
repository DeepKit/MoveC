# C盘瘦身工具 (DiskCleanup)

## 📖 项目简介

C盘瘦身工具是一个专门用于分析和优化Windows系统C盘空间的实用工具。通过智能分析目录的符号链接可行性，帮助用户安全地将大文件和目录移动到其他磁盘，并创建符号链接保持程序正常运行。

## ✨ 主要特性

### 🔍 智能分析
- **目录可行性分析**: 基于目录类型和位置智能判断移动风险
- **三级风险评估**: 可链接、有风险、禁止移动
- **快速响应**: 基于路径分析，无需遍历大量文件

### 📊 目录大小计算
- **完整递归计算**: 包含所有子目录和文件的真实大小
- **智能根目录保护**: 自动检测并警告大目录操作
- **格式化显示**: 自动转换GB/MB/KB/字节单位

### 🎨 用户界面
- **彩色状态显示**: RichEdit支持的彩色文本反馈
- **即时响应**: 目录浏览和文件显示无延迟
- **功能分离**: 浏览、计算、分析功能独立操作

### 🛡️ 安全机制
- **根目录保护**: 防止对系统关键目录的误操作
- **异常处理**: 完善的错误处理和资源清理
- **用户确认**: 重要操作前的明确提示

## 🚀 快速开始

### 系统要求
- Windows 7/8/10/11
- 管理员权限（推荐）
- 至少100MB可用磁盘空间

### 安装步骤
1. 下载最新版本的DiskCleanup.exe
2. 以管理员身份运行程序
3. 开始使用各项功能

### 基本使用流程
1. **浏览目录**: 双击左侧目录树选择要分析的目录
2. **计算大小**: 点击"计算目录大小"按钮了解空间占用
3. **分析可行性**: 点击"分析并标记"按钮评估移动风险
4. **执行操作**: 根据分析结果决定是否移动目录

## 📋 功能详解

### 目录浏览
- **双击切换**: 双击目录树快速切换并显示文件
- **路径同步**: 自动更新对应的路径编辑框
- **文件列表**: 实时显示目录中的文件内容

### 大小计算
- **递归计算**: 包含所有子目录的完整大小统计
- **性能优化**: 对根目录等大目录给出性能警告
- **详细显示**: 在标签和状态区域显示格式化结果

### 可行性分析
- **智能判断**: 基于目录性质判断符号链接可行性
- **风险分级**: 
  - 🟢 **可链接**: 安全移动，不影响程序运行
  - 🟡 **有风险**: 可能影响程序，但符号链接通常可解决
  - 🔴 **禁止移**: 严重影响系统，禁止移动
- **详细说明**: 提供具体的判断原因和操作建议

## 🛡️ 安全注意事项

### 使用前须知
1. **备份重要数据**: 在执行任何移动操作前，请备份重要文件
2. **管理员权限**: 建议以管理员身份运行，确保有足够权限
3. **系统目录**: 绝对不要移动Windows系统目录
4. **测试环境**: 重要系统建议先在测试环境验证

### 风险提示
- **符号链接**: 某些老旧程序可能不支持符号链接
- **权限问题**: 移动系统文件可能导致权限问题
- **路径依赖**: 硬编码路径的程序可能受到影响
- **性能影响**: 跨磁盘的符号链接可能影响访问速度
