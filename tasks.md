## 🚀 快速开始

### 系统要求
- Windows 7/8/10/11
- 管理员权限（推荐）
- 至少100MB可用磁盘空间

### 安装步骤
1. 下载最新版本的DiskCleanup.exe
2. 以管理员身份运行程序
3. 开始使用各项功能

### 基本使用流程
1. **浏览目录**: 双击左侧目录树选择要分析的目录
2. **计算大小**: 点击"计算目录大小"按钮了解空间占用
3. **分析可行性**: 点击"分析并标记"按钮评估移动风险
4. **执行操作**: 根据分析结果决定是否移动目录

## 📋 功能详解

### 目录浏览
- **双击切换**: 双击目录树快速切换并显示文件
- **路径同步**: 自动更新对应的路径编辑框
- **文件列表**: 实时显示目录中的文件内容

### 大小计算
- **递归计算**: 包含所有子目录的完整大小统计
- **性能优化**: 对根目录等大目录给出性能警告
- **详细显示**: 在标签和状态区域显示格式化结果

### 可行性分析
- **智能判断**: 基于目录性质判断符号链接可行性
- **风险分级**: 
  - 🟢 **可链接**: 安全移动，不影响程序运行
  - 🟡 **有风险**: 可能影响程序，但符号链接通常可解决
  - 🔴 **禁止移**: 严重影响系统，禁止移动
- **详细说明**: 提供具体的判断原因和操作建议
