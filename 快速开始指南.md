# 🚀 快速开始指南 - AES-256升级

**版本**: v1.0.0 → v1.2.0  
**升级时间**: 约10分钟  
**难度**: ⭐⭐ 简单

---

## ✅ 5步完成升级

### 步骤1：备份数据库（1分钟）

```bash
copy MoveC.db MoveC.db.backup
```

✅ 完成后打勾：[ ]

---

### 步骤2：配置Release编译选项（2分钟）

1. 打开项目选项
2. 找到 **Delphi Compiler** → **Compiling**
3. 在 **Conditional defines** 中添加：`RELEASE`
4. 保存设置

✅ 完成后打勾：[ ]

---

### 步骤3：编译ImportImages工具（2分钟）

```bash
dcc32 ImportImages.dpr
```

或双击运行：`compile_importimages.bat`

✅ 完成后打勾：[ ]

---

### 步骤4：重新导入图像（3分钟）

```bash
ImportImages.exe
```

**预期输出**：
```
=== 图像数据导入工具 (AES-256加密版) ===
防篡改包初始化完成 - 使用AES-256加密
成功导入图像: wechat (12000 字节)
成功导入图像: alipay (15000 字节)
...
图像导入完成
```

✅ 完成后打勾：[ ]

---

### 步骤5：测试验证（2分钟）

1. 编译主程序（Debug和Release版本）
2. 运行程序
3. 打开打赏页面
4. 确认所有图像显示正常

✅ 完成后打勾：[ ]
