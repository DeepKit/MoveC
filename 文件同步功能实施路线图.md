## 🎯 V1.4.0 文件同步功能概览

### 核心价值主张
> **为开发者和内容创作者提供可靠的本地/局域网文件自动同步解决方案**

### 目标用户
1. **开发者**: 代码项目在多台电脑/硬盘间同步
2. **内容创作者**: 文档、设计稿在本地和NAS间备份
3. **企业用户**: 部门共享文件夹实时同步

### 关键特性
✅ **最多 6 组同步任务** - 满足多项目/多场景需求  
✅ **实时 + 手动双模式** - 灵活适应不同使用场景  
✅ **智能过滤规则** - 预设模板 + 自定义规则  
✅ **本地 + 局域网支持** - 不依赖互联网  
✅ **SQLite3 持久化** - 配置和状态可靠存储  
✅ **开发者友好预设** - Delphi/Python/Node.js 项目优化

### 核心组件
```
┌─────────────────────────────────────────────────────────────┐
│                       MoveC 主程序                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ┌──────────────┐  ┌──────────────┐  ┌──────────────┐    │
│  │  同步引擎    │  │  文件监控    │  │  预设管理    │    │
│  │ TSyncEngine  │  │ TFSWatcher   │  │ TPresetMgr   │    │
│  └──────────────┘  └──────────────┘  └──────────────┘    │
│         │                 │                  │             │
│         └─────────────────┴──────────────────┘             │
│                           │                                 │
│                  ┌────────▼────────┐                       │
│                  │  SQLite3 数据库  │                       │
│                  │  - sync_tasks    │                       │
│                  │  - sync_presets  │                       │
│                  │  - sync_history  │                       │
│                  │  - sync_file_states │                    │
│                  └──────────────────┘                       │

```

### 数据库表结构
- **sync_tasks**: 同步任务配置（源/目标路径、模式、状态）
- **sync_presets**: 预设模板（过滤规则、系统/用户预设）
- **sync_history**: 同步历史记录（统计信息、错误日志）
- **sync_file_states**: 文件状态跟踪（哈希、时间戳）

## 🎓 预设模板说明

### 1. 开发代码同步（推荐）
**适用场景**: Delphi/Python/Node.js 项目开发

**包含规则**:
```
包含: *.pas, *.dpr, *.dfm, *.py, *.js, *.ts, *.json, *.md, *.txt
排除: __pycache__/*, node_modules/*, *.dcu, *.pyc, *.o, 
      Win64/*, Win32/*, Debug/*, Release/*, .git/*, .svn/*
```

**特点**:
- ✅ 仅同步源代码和配置文件
- ✅ 自动排除编译产物和依赖库
- ✅ 保留版本控制元数据（.git 除外）

---

### 2. 文档备份
**适用场景**: Office 文档、PDF 备份

**包含规则**:
```
包含: *.doc, *.docx, *.xls, *.xlsx, *.ppt, *.pptx, 
      *.pdf, *.txt, *.rtf, *.odt, *.ods, *.odp
排除: ~$*, *.tmp, *.bak
```

**特点**:
- ✅ 涵盖所有常见文档格式
- ✅ 自动排除临时文件和锁文件
- ✅ 适合定期手动同步

---

### 3. 全量同步（谨慎使用）
**适用场景**: 完整目录镜像

**包含规则**:
```
包含: *
排除: （无）
```

**警告**:
- ⚠️ 会同步所有文件，包括系统文件
- ⚠️ 可能占用大量时间和空间
- ⚠️ 建议仅用于小型目录

## 🔧 故障排查指南

### 问题 1: 实时同步不工作
**症状**: 文件修改后未自动同步

**排查步骤**:
1. 检查任务状态是否为"运行中"
2. 查看日志是否有错误信息
3. 确认源目录路径可访问
4. 验证过滤规则是否排除了该文件

**解决方案**:
- 重启任务
- 调整过滤规则
- 检查文件系统权限

---

### 问题 2: 局域网同步失败
**症状**: 提示"网络路径不可访问"

**排查步骤**:
1. 手动访问网络路径（\\192.168.x.x\share）
2. 检查网络连接状态
3. 验证共享文件夹权限
4. 测试 ping 网络主机

**解决方案**:
- 确保网络畅通
- 配置正确的用户名/密码
- 检查防火墙设置

---

### 问题 3: 同步速度慢
**症状**: 传输速度远低于预期

**排查步骤**:
1. 检查是否有其他程序占用带宽
2. 查看 CPU 和磁盘使用率
3. 确认是否在同步大量小文件
4. 检查是否启用了压缩传输

**优化建议**:
- 减少并发任务数
- 调整过滤规则，排除不必要的文件
- 升级网络设备（千兆网卡/交换机）
