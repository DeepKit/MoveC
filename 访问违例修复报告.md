**错误类型**: Access Violation - 内存访问违例  
**修复结果**: 程序安全退出，无访问违例错误  

## 🔍 问题分析

### 错误详情
```
Access violation at address 0071A268 in module 'C盘瘦身.exe'
(offset A268). Read of address 0201C794.
```

### 根本原因
1. **对象生命周期问题**: 在对象销毁过程中访问已释放的内存
2. **事件处理时机**: 在窗体销毁时仍然触发事件处理
3. **资源清理不当**: FormDestroy中没有正确处理资源清理
4. **异常处理缺失**: 退出过程中缺少异常保护

### 触发场景
- 点击【退出】按钮时
- 窗体关闭过程中
- ShellTreeView事件在销毁时触发
- 状态栏更新时访问已释放的对象

## 🛠️ 修复方案

### 1. 安全的退出处理
```pascal
// 修复前（不安全的退出）
procedure TfrmMain.btnExitClick(Sender: TObject);
begin
  if FIsProcessing then
  begin
    if MessageDlg('操作正在进行中，确定要退出吗？',
                  mtConfirmation, [mbYes, mbNo], 0) = mrNo then
      Exit;
  end;
  Close;  // 直接关闭，可能导致访问违例
end;

// 修复后（安全的退出）
procedure TfrmMain.btnExitClick(Sender: TObject);
begin
  try
    // 检查是否有操作正在进行
    if FIsProcessing then
    begin
      if MessageDlg(_('confirm_exit'), mtConfirmation, [mbYes, mbNo], 0) = mrNo then
        Exit;
      
      // 停止正在进行的操作
      FIsProcessing := False;
    end;

    // 安全关闭窗体
    Close;
  except
    on E: Exception do
    begin
      // 如果关闭过程中出现异常，强制退出
      try
        UpdateStatus('退出时发生错误: ' + E.Message);
      except
        // 忽略状态更新错误
      end;
      
      // 强制退出应用程序
      Application.Terminate;
    end;
  end;
end;
```

### 3. 安全的事件处理
```pascal
// 修复前（不安全的事件处理）
procedure TfrmMain.stvSourceChange(Sender: TObject; Node: TTreeNode);
begin
  try
    if Assigned(Node) then
    begin
      SelectedPath := stvSource.Path;  // 可能访问已释放的对象
      FSourcePath := SelectedPath;
      edtSourceDir.Text := SelectedPath;  // 可能访问已释放的控件
    end;
  except
    on E: Exception do
      UpdateStatus('源目录选择出错: ' + E.Message);
  end;
end;

// 修复后（安全的事件处理）
procedure TfrmMain.stvSourceChange(Sender: TObject; Node: TTreeNode);
begin
  try
    // 检查组件和节点是否有效
    if Assigned(stvSource) and Assigned(Node) and not (csDestroying in ComponentState) then
    begin
      SelectedPath := stvSource.Path;
      if SelectedPath <> '' then
      begin
        FSourcePath := SelectedPath;
        
        // 安全更新编辑框
        if Assigned(edtSourceDir) then
          edtSourceDir.Text := SelectedPath;
          
        UpdateStatus(_('source_selected') + SelectedPath);
        
        // 安全更新状态栏
        if Assigned(StatusBar1) and (StatusBar1.Panels.Count > 1) then
          StatusBar1.Panels[1].Text := _('status_source') + ExtractFileName(SelectedPath);
      end;
    end;
  except
    on E: Exception do
    begin
      try
        UpdateStatus('源目录选择出错: ' + E.Message);
      except
        // 忽略状态更新错误
      end;
    end;
  end;
end;
```
