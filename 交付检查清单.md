# 交付检查清单

以下清单用于发布前最终确认。建议按章节逐项勾选。

## 一、应用功能与体验
- [ ] 启动与 About 页面：5 幅图片正常显示，切换无卡顿。
- [ ] 主要功能点：体积分析、智能迁移、创建目录链接、回滚备份可用。
- [ ] 错误提示与引导：权限不足、路径错误、空间不足等具备清晰提示。
- [ ] 兼容性：在 Windows 10/11 测试通过（如需覆盖 Win7，请额外验证）。

## 二、数据库与数据
- [ ] 表名仍为 `images`（已通过约束脚本重建为受约束版本）。
- [ ] 约束脚本 `UpgradeDatabase_Constraints.sql` 已在最终发版库上执行。
- [ ] 回填记录完整可读（5 条示例记录），抽查图像展示正常。

## 三、日志与配置
- [ ] Release 构建下，默认仅记录 WARN/ERROR 日志（无需 ini）。
- [ ] 如需临时排障，设置环境变量 `MOVEC_LOG_LEVEL=DEBUG` 可生效（不做永久配置）。
- [ ] 运行目录无多余临时日志（log、*_output.txt 等）。
