# MoveC 发布任务（tasks）

更新时间：2025-10-23 11:31

## 发布就绪度结论（C盘瘦身）
- 当前不满足“可发布”标准：
  - 目录迁移为演示级流程，缺少校验、断点续迁、完善回滚与占用处理。
  - 智能重复文件清理模块未完成（存在未定义引用，无法编译）。
  - 清理功能主体已具备，但缺少服务控制、预览与更严格保护。

## 必须项（发布前）
- [ ] 目录迁移（从演示到生产）
  - [ ] 拷贝后进行 SHA-256 校验（逐文件/抽样）并汇总报告
  - [ ] 拷贝失败或校验失败立即停止并回滚
  - [ ] 创建 Junction 后 VerifyJunction 审计（源→目标一致性、可访问性）
  - [ ] 事务状态持久化（INI/JSON）：started/progress/done/failed/rolled_back
  - [ ] 断点续迁：检测未完成事务后可恢复或回滚
  - [ ] 回滚流程：删除联接→迁回文件→可选清空目标（送回收站）
  - [ ] 取消支持：串联 FCancelRequested，全链路尊重取消
  - [ ] 权限/占用：管理员权限检测；文件占用检测与提示（可选 Restart Manager）
  - [ ] 迁移后引导：一键“删除备份目录以真正释放空间”并显示预计可释放容量
- [ ] 智能重复文件清理（最小可用）
  - [ ] 修复 uDuplicateFiles 编译错误：补充 FDetector、FDuplicateGroups 或改为内置扫描器
  - [ ] 实现三阶段检测：大小分桶→部分哈希(64KB)→全量 SHA-256
  - [ ] UI 展示分组与可节省空间；支持勾选与“推荐选择”
  - [ ] 清理策略：同卷优先硬链接替换，跨卷移回收站；错误处理与回滚
  - [ ] 并发与进度：TParallel.For + 临界区；进度/剩余时长估算；可取消
- [ ] 清理功能（强化）
  - [ ] Windows Update 缓存清理前停止相关服务（wuauserv、bits），完成后恢复
  - [ ] 清理前预览：将要删除的路径与大小，用户确认后再执行
  - [ ] IsSafeToDelete 白名单/黑名单再强化，避免误删（扩展文件类型与目录规则）
  - [ ] 生成清理报告并保存历史
- [ ] UI/体验与安全
  - [ ] 去除 Emoji（标题/按钮/状态文本），统一字体与样式
  - [ ] 关键操作二次确认与风险提示
  - [ ] 统一日志开关（Debug/Release）与文件落盘
