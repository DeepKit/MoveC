# C盘超级瘦身 - 宣传网站

这是C盘超级瘦身工具的官方宣传网站，采用现代化设计，提供完整的产品介绍和使用指南。

## 🎨 设计特点

### 视觉效果
- **深色主题**：采用深色配色方案，现代感十足，减少眼睛疲劳
- **渐变动画**：标题和按钮使用渐变色，增强视觉吸引力
- **卡片悬停效果**：鼠标悬停时卡片上浮并显示顶部渐变条
- **平滑滚动**：页面内锚点跳转使用平滑滚动动画
- **视差效果**：Hero区域具有轻微的视差滚动效果

### 交互功能
- **返回顶部按钮**：滚动超过300px自动显示，点击平滑返回顶部
- **导航高亮**：根据当前浏览位置自动高亮对应的导航链接
- **渐入动画**：卡片和内容区块滚动到视口时自动淡入
- **数字计数器**：统计数据区域的数字会动态递增显示
- **FAQ折叠面板**：常见问题采用可展开/折叠的设计

### 响应式设计
- **桌面端**：1100px最大宽度，功能卡片4列布局
- **平板端**：768px以下，卡片变为2列布局，导航居中
- **移动端**：480px以下，卡片单列布局，字体和间距优化

## 🎯 使用方法

### 本地预览
1. 直接在浏览器中打开 `index.html` 文件
2. 或使用本地服务器：
   ```bash
   # 使用 Python
   python -m http.server 8000
   
   # 使用 Node.js
   npx serve
   ```
3. 访问 `http://localhost:8000`

### 部署到服务器
1. 将整个 `html/` 目录上传到 Web 服务器
2. 确保服务器支持静态文件托管
3. 配置域名指向该目录

## 🔧 自定义修改

### 修改颜色主题
在 `styles.css` 的 `:root` 部分修改CSS变量：
```css
:root {
  --bg: #0b0f1a;           /* 背景色 */
  --fg: #e6edf3;           /* 前景色/文字色 */
  --primary: #6aa7ff;      /* 主色调 */
  --card: #121829;         /* 卡片背景 */
  --border: #1f2640;       /* 边框颜色 */
}
```

### 修改统计数据
在 `index.html` 中找到 `#stats` 区域，修改 `data-count` 属性：
```html
<span class="stat-number" data-count="10000">10,000</span>
```

### 替换占位图
将 `assets/` 目录中的SVG占位图替换为实际的截图（支持PNG、JPG、SVG格式）
