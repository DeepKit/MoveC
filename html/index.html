常见问题

迁移是否安全？如何回滚？

迁移前会完整复制并备份原目录（带时间戳后缀），再创建目录链接以保持路径不变。需要回滚时，删除链接并把备份目录改回原名即可。软件提供完整的操作日志，每一步都可追溯。

是否需要管理员权限？

创建目录链接通常需要管理员权限。若提示权限不足，请以管理员身份运行软件。软件会在需要时自动提示您提升权限。

会影响已安装的软件吗？

软件通过创建目录链接保持原路径不变，理论上不影响已安装程序。但迁移完成后请按提示进行验证测试，确保相关程序正常运行。建议先迁移非关键目录进行测试。

为什么需要完整性校验？

完整性校验可防止资源被篡改或替换，保护你的使用体验与安全。当校验失败时，软件会阻断运行并引导访问官网获取正版资源。这是对用户负责的安全措施。

支持哪些 Windows 版本？

支持 Windows 7 及以上版本（包括 Windows 10、Windows 11）。建议使用 Windows 10 或更高版本以获得最佳体验。

可以迁移哪些类型的目录？

可以迁移用户文档、下载文件夹、桌面、图片、视频、游戏数据等各类目录。不建议迁移系统关键目录（如 Windows、Program Files 等）。
