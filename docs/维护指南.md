# MoveC 维护指南

## 编译环境

### 必需软件
- **IDE**: Embarcadero RAD Studio 12.0 (Delphi 12 Athens) 或更高版本
- **编译器**: dcc64.exe (Win64) / dcc32.exe (Win32)
- **数据库**: SQLite (FireDAC 内置驱动)

### 编译命令

#### Win64 Release
```powershell
& "d:\Program Files (x86)\Embarcadero\Studio\23.0\bin\dcc64.exe" `
  -Q -B `
  "-NSSystem;Winapi;Vcl;System.Win;Vcl.Imaging;Vcl.Shell;Data;Data.Win;FireDAC;FireDAC.Comp;FireDAC.DApt;FireDAC.Phys;FireDAC.Stan" `
  "-UD:\Program Files (x86)\Embarcadero\Studio\23.0\lib\Win64\release;AntiTamperPackage;syncLocal" `
  "-NWin64\Release" `
  "-EWin64\Release" `
  "C盘超级瘦身.dpr"
```

#### Win32 Release
```powershell
& "d:\Program Files (x86)\Embarcadero\Studio\23.0\bin\dcc32.exe" `
  -Q -B `
  "-NSSystem;Winapi;Vcl;System.Win;Vcl.Imaging;Vcl.Shell;Data;Data.Win;FireDAC;FireDAC.Comp;FireDAC.DApt;FireDAC.Phys;FireDAC.Stan" `
  "-UD:\Program Files (x86)\Embarcadero\Studio\23.0\lib\Win32\release;AntiTamperPackage;syncLocal" `
  "-NWin32\Release" `
  "-EWin32\Release" `
  "C盘超级瘦身.dpr"
```

## 数据库

### 文件
- `MoveC.db` - SQLite 数据库，运行时自动创建

### 主要表
- `images` - 防篡改图像存储
- `migrations` - 迁移事务记录（如有）

### 配置文件
- `CleanupScheduler.ini` - 清理任务配置
- `MigrationScheduler.ini` - 迁移任务配置
- `cleanup_history.json` - 清理历史记录

## 发布打包

### Release 编译注意事项
1. 确保定义 `RELEASE` 编译符号
2. 禁用详细日志 (`NO_DEBUG_LOG`)
3. 启用反调试保护

### 发布包内容
```
C盘瘦身神器_V1.4.0/
├── C盘超级瘦身.exe
├── README.txt
├── 用户手册.pdf (可选)
└── MoveC.db (首次运行自动生成)
```

## 常见问题

### Q: 编译报错找不到单元
A: 检查 `-U` 参数中的搜索路径，确保包含 `AntiTamperPackage` 和 `syncLocal`

### Q: 运行时提示数据库缺失
A: 使用 SeedTool.exe 播种或让程序自动创建

### Q: 防篡改校验失败
A: 重新运行 SeedTool.exe 播种数据库
