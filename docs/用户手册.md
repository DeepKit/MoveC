## 软件简介

C盘超级瘦身是一款专为 Windows 用户设计的磁盘空间管理工具，主要功能包括：

- **目录迁移**: 将大体积目录从 C 盘迁移到其他磁盘，通过目录链接保持原路径可用
- **智能清理**: 清理回收站、临时文件、浏览器缓存、Windows 更新缓存等
- **重复文件清理**: 扫描并清理重复文件，释放磁盘空间
- **清理历史**: 记录每次清理操作，支持导出报告

## 安装与启动

### 系统要求

- **操作系统**: Windows 7/8/10/11 (32位或64位)
- **权限**: 部分功能需要管理员权限
- **磁盘空间**: 至少 50 MB 可用空间

### 启动方式

1. 双击 `C盘超级瘦身.exe` 启动程序
2. 如需使用迁移功能，请右键选择"以管理员身份运行"

### 管理员权限说明

以下功能需要管理员权限：
- 目录迁移（创建目录链接）
- Windows 更新缓存清理（需停止系统服务）
- 预取文件清理
- 系统日志清理

## 功能说明

### 目录迁移

将指定目录从 C 盘迁移到其他磁盘，同时创建目录链接保持原路径可用。

**适用场景**:
- 用户文档、下载、桌面等大体积目录
- 游戏数据目录
- 应用程序缓存目录

**迁移流程**:
1. 选择源目录（C 盘上的目录）
2. 选择目标位置（其他磁盘）
3. 程序自动：复制文件 → 备份原目录 → 创建目录链接
4. 验证迁移结果

**注意事项**:
- 迁移前会自动备份原目录（带时间戳后缀）
- 迁移后原路径仍可正常访问
- 不建议迁移系统关键目录（Windows、Program Files 等）

### 智能清理

一键清理多种可释放空间的内容。

**清理项目**:

| 清理项 | 说明 | 风险等级 |
|--------|------|----------|
| 回收站 | 清空 Windows 回收站 | 低 |
| 临时文件 | 用户和系统临时目录 | 低 |
| 浏览器缓存 | Chrome/Edge/Firefox/IE 缓存 | 低 |
| Windows 更新缓存 | SoftwareDistribution 目录 | 中 |
| 系统日志 | Windows 日志和调试文件 | 中 |
| 预取文件 | Prefetch 目录 | 低 |

**清理预览**:
- 清理前可预览所有待清理项目
- 按风险等级显示不同颜色（绿色=安全，黄色=注意，红色=高风险）
- 支持按类型、大小、时间排序
- 可选择性清理（勾选/取消勾选）

### 重复文件清理

扫描并清理重复文件，采用三阶段检测：

1. **大小分组**: 按文件大小快速分组
2. **部分哈希**: 读取文件前后各 64KB 计算哈希
3. **全量哈希**: SHA-256 完整校验确认

**操作步骤**:
1. 选择扫描目录
2. 点击"开始扫描"
3. 查看重复文件组
4. 选择要删除的文件（保留一份）
5. 确认删除

### 清理历史

查看历史清理记录，支持导出报告。

**功能**:
- 按时间、类型、状态查看历史记录
- 统计累计释放空间和删除文件数
- 导出 TXT 文本报告
- 导出 JSON 格式数据

**访问方式**: 菜单 → 清理 → 清理历史记录

## 操作指南

### 快速清理（推荐新手）

1. 启动程序
2. 点击菜单"清理" → "智能清理"
3. 在预览窗口查看待清理项目
4. 确认后点击"执行清理"
5. 等待清理完成

### 目录迁移步骤

1. 以管理员身份启动程序
2. 在左侧目录树中选择要迁移的目录
3. 在右侧选择目标磁盘位置
4. 点击"开始迁移"
5. 等待迁移完成，验证原路径是否正常

### 清理 Windows 更新缓存

1. 以管理员身份启动程序
2. 点击菜单"清理" → "清理更新缓存"
3. 程序会自动停止 Windows Update 服务
4. 清理完成后自动恢复服务

## 回滚与恢复

### 迁移回滚

如果迁移后出现问题，可以回滚到原始状态：

**方法一：使用程序回滚**
1. 点击菜单"操作" → "一键回退"
2. 选择要回滚的迁移记录
3. 确认回滚

**方法二：手动回滚**
1. 删除目录链接（原位置）
2. 将备份目录（带时间戳后缀）重命名为原目录名
3. 完成

### 备份目录位置

迁移时创建的备份目录命名规则：
```
原目录名_backup_YYYYMMDD_HHMMSS
```

例如：`Documents_backup_20251202_143022`

### 清理恢复

- **回收站清理**: 无法恢复，清理前请确认
- **临时文件清理**: 通常不影响系统运行
- **浏览器缓存**: 浏览器会自动重建缓存
- **更新缓存**: Windows 会在需要时重新下载

## 常见问题

### Q: 迁移后程序无法正常运行？

**A**: 
1. 检查目录链接是否创建成功（应显示为带箭头的快捷方式图标）
2. 确认目标磁盘有足够空间且未断开
3. 尝试使用回滚功能恢复

### Q: 清理后系统变慢？

**A**: 
- 预取文件清理后，首次启动程序会略慢，系统会自动重建
- 浏览器缓存清理后，网页加载可能暂时变慢

### Q: 需要管理员权限但无法获取？

**A**: 
1. 右键程序 → "以管理员身份运行"
2. 如果仍无法获取，联系系统管理员

### Q: 清理时提示文件被占用？

**A**: 
1. 关闭可能占用文件的程序（浏览器、Office 等）
2. 重新执行清理
3. 部分系统文件无法清理是正常现象

### Q: 如何查看释放了多少空间？

**A**: 
1. 菜单 → 清理 → 清理历史记录
2. 查看累计释放空间统计
3. 或导出报告查看详细信息

## 安全说明

### 受保护的系统路径

程序会自动保护以下关键路径，防止误删除：

- `C:\Windows\System32`
- `C:\Windows\SysWOW64`
- `C:\Windows\Boot`
- `C:\Windows\Fonts`
- `C:\Windows\WinSxS`
- `C:\Program Files`
- `C:\Program Files (x86)`
- `C:\Users\Default`
- `C:\Users\Public`
- 用户配置文件（NTUSER.DAT 等）

### 安全删除白名单

只有以下类型的文件会被清理：
- 临时文件（.tmp, .temp）
- 日志文件（.log）
- 缓存文件（.cache）
- 备份文件（.bak, .old）
- 以 ~ 开头的临时文件

### 数据安全建议

1. **首次使用前备份重要数据**
2. **清理前使用预览功能确认**
3. **定期导出清理历史报告**
4. **不要手动删除迁移备份目录**（除非确认不再需要回滚）
