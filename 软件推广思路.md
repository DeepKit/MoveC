# 软件推广思路（C盘超级瘦身）

## 目标人群
- 普通 Windows 用户：系统盘空间紧张、电脑变慢、不会手动迁移目录的人群。
- 游戏玩家与重度下载用户：下载/游戏更新导致 C 盘爆满。
- 轻量办公人群：希望快速释放空间，提升办公效率。

## 核心卖点（用户语言）
- 一键释放系统盘空间：可视化操作，三步完成。
- 智能迁移建议：自动评估体积，给出优先级与建议路径。
- 安全回滚：迁移前自动备份原目录；如不满意，原路恢复。
- 速度快、占用低：原生 Windows 程序，安装小、启动快。
- 完全免费、无广告、无捆绑。

## 标准素材与物料
- 官方网站（html/）：首页、功能亮点、常见问题、下载按钮（已优化为用户导向）。
- 截图素材：
  - 首页/主界面
  - 体积分析页
  - 迁移向导页
  - 成功页（显示释放空间）
- 15～30 秒短视频：演示“三步走”流程（下载安装 → 一键扫描 → 智能迁移）。
- 文案模板：
  - 标题：C 盘不够用？一键释放系统盘空间
  - 副标题：智能迁移 + 安全回滚，电脑更快更稳
  - 行动按钮：立即下载 / 了解如何使用

## 推广渠道与打法
- 应用商店/软件平台：
  - 国内：360 软件管家、腾讯电脑管家、华军软件园、太平洋下载、ZOL 下载、吾爱破解论坛（仅宣传版块）等。
  - 海外（可选）：Softpedia、MajorGeeks。
- 社媒与内容：
  - B站/抖音/快手：发布 15～30 秒操作演示；简介放下载链接（或官网）。
  - 小红书/知乎：图文教程（如何一键释放空间、迁移前后对比、常见问题）。
  - 微信公众号：发布使用指引与版本更新公告。
- 垂直社区：
  - Steam、WeGame 等游戏平台相关社区（话题：更新占用大、如何把游戏迁到 D 盘）。
  - 数码/办公类贴吧与论坛（如 V2EX、少数派、酷安）。
- 搜索引擎与SEO（中期）：
  - 关键词：C盘空间不足、释放系统盘空间、移动用户目录、迁移游戏下载目录等。
  - 在官网部署结构化数据与站点地图（可后续安排）。

## 活动与增长设计
- 新手任务：首次使用引导提示“3步完成”，降低第一次操作成本。
- 小成就反馈：迁移成功后展示释放空间数值与预计提速提示，可一键复制分享（生成简图）。
- 轻口碑机制：
  - 在官网与应用内“反馈入口”，收集用户评价；优质评价用于展示（征得许可）。
  - 定期发布“版本更新日志”，建立信任感。

## 转化路径建议
1) 用户看到短视频/图文 → 点击进入官网。
2) 官网 5 秒内明确“解决什么问题、如何开始”。
3) 下载、安装、首次运行（提供一键扫描按钮）。
4) 一次成功体验 → 引导关注公众号/收藏官网，便于更新与反馈。

## FAQ（面向用户）建议补充项
- 需要管理员权限吗？如何以管理员运行？
- 迁移失败如何处理？如何查看日志定位？
- 如果空间仍不够，推荐清理哪些目录？
- 迁移了游戏/下载目录后，平台是否正常？如何验证？

## 版本节奏
- v1：功能完备 + 官网 + 基础素材 + 多平台分发。
- v1.x：根据反馈优化引导、完善 FAQ、补充模板脚本（如快速定位大文件）。
- v2：补充更多自动化能力（如一键清理策略、更多目录模板）。
