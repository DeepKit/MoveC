## 🐛 常见问题

### 问题1: 任务ID始终为0

**原因：** 数据库写入失败

**调试步骤：**
```
1. 检查数据库文件权限
2. 查看 DebugView 输出
3. 检查是否有异常日志
```

**解决方法：**
```
1. 以管理员身份运行程序
2. 删除 MoveC.db 让程序重新创建
3. 检查磁盘空间是否充足
```

---

### 问题2: 重启后任务消失

**原因：** 加载失败或数据库连接问题

**调试步骤：**
```
1. 查看启动时的日志输出
2. 检查"已加载 X 个同步任务"数量
3. 查看 DebugView 的 LoadTasksFromDatabase 输出
```

**解决方法：**
```
1. 确认数据库文件存在
2. 检查数据库连接是否成功
3. 查看数据库中是否有数据
```

---

### 问题3: 数据库连接失败

**原因：** 文件路径错误或权限问题

**调试步骤：**
```
1. 查看日志中的数据库路径
2. 检查该路径是否可写
3. 检查是否被杀毒软件拦截
```

**解决方法：**
```
1. 以管理员身份运行
2. 将程序安装到非系统目录
3. 添加杀毒软件白名单
```
