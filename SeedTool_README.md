# 防篡改播种工具使用说明

## 功能概述

`SeedTool.exe` 是一个独立的图像资源加密播种工具，用于将图像文件加密后写入 `MoveC.db` 数据库。

## 安全机制

- **KDF 密钥派生**：使用 PBKDF2 (SHA-256) 从管理员密码派生加密密钥
- **AES-256 加密**：图像数据使用派生密钥进行 AES-256 加密
- **双重校验**：
  - SHA-256：验证数据完整性
  - HMAC-SHA256：防止数据被篡改和伪造

## 使用步骤

### 1. 启动播种工具
```
运行 SeedTool.exe
```

### 2. 输入管理员密码
- 密码长度不少于 8 位
- **重要**：此密码必须与主程序 `uMain.pas` 中的 `Config.EncryptionKey` 一致
- 当前主程序使用的密码：`MoveC_AntiTamper_Key_2025`

### 3. 添加图像文件
- 点击"添加图像"按钮
- 选择一个或多个图像文件（支持 PNG、JPG、BMP、GIF 等）
- 图像的 key 将自动使用文件名（不含扩展名）

### 4. 开始播种
- 点击"开始播种"按钮
- 工具将：
  1. 初始化防篡改包（使用输入的密码）
  2. 连接数据库 `MoveC.db`
  3. 创建/验证表结构（包含 `sha256_hash` 和 `hmac_sha256` 字段）
  4. 逐个加密并写入图像

### 5. 查看日志
- 播种过程的详细日志会显示在底部日志区域
- 包括成功/失败状态

## 注意事项

1. **密码一致性**
   - 播种工具使用的密码必须与主程序一致
   - 否则主程序无法解密图像

2. **数据库位置**
   - 数据库文件 `MoveC.db` 应与 `SeedTool.exe` 在同一目录
   - 或使用主程序的数据库路径

3. **严格模式**
   - 播种工具使用严格模式
   - 所有记录都包含 `sha256_hash` 和 `hmac_sha256` 字段

4. **主程序配置**
   - 主程序必须启用相同的配置：
     ```pascal
     Config.Salt := 'MoveC_Salt_v1';
     Config.KdfIterations := 10000;
     Config.EnableHMAC := True;
     ```

## 安全建议

1. **管理员密码**
   - 使用强密码（建议 16 位以上，包含大小写字母、数字、特殊字符）
   - 妥善保管密码，不要泄露

2. **播种环境**
   - 在安全的环境中运行播种工具
   - 播种完成后立即关闭工具

3. **数据库保护**
   - 播种后的 `MoveC.db` 应妥善保护
   - 定期备份

## 故障排除

### 播种失败
- 检查密码长度是否符合要求
- 检查图像文件是否存在
- 查看日志区域的详细错误信息

### 主程序无法解密
- 确认播种工具和主程序使用相同的密码
- 确认 Salt、KdfIterations、EnableHMAC 配置一致

### 数据库错误
- 检查 `MoveC.db` 文件是否存在
- 检查文件权限
- 尝试使用 `MoveC.reset` 清空并重新播种

## 与主程序集成

主程序（`C盘超级瘦身.exe`）会：
1. 初始化相同的防篡改配置
2. 读取并解密图像数据
3. 验证 SHA-256 和 HMAC-SHA256
4. 如果校验失败，触发安全退出（fail-closed）
